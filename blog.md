# 10月17日笔记
--- 
### 常用命令
- shutdown  -r  时间  重启(shutdown -c &nbsp 命令取消重启)
- shutdown  -h  时间  关机
- ls  显示文件和目录列表
- mkdir  创建文件夹
- touch  创建文件
- cp  复制文件
- du  统计目录及文件空间占用情况
- file  查看文件类型
- rm  删除文件或文件夹
- mv  移动文件或重命名
- which  显示系统命令所在目录
 
---	
### 其他内容
- vi etc/hostname  修改主机名
- ping &nbsp [选项] 目标主机  测试网络连通性
- yum install net-tools  下载命令插件
